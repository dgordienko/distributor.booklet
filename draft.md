# Цель
Создать приложение (фронтенд и бэкенд) для торгового представителя которое позволит предстоставлять контент (фотографии, описания) товаров для продажи в реальной торговой точке на экране планшета

# Бэкенд/Фронтенд сервер
Приложение, которое позволит редактировать информацию о товаре в текстовом виде, добавлять фотографии и описания

# Фронтенд
Приложение, которое позволит пользователю  просматривать товары и их информацию на экране планшета

# Технологии
- React
- Node.js
- Express
- Kotlin (Android)
